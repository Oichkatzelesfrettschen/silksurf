# Code of Conduct

The silksurf project adopts the **Contributor Covenant, version 2.1**, as
its code of conduct. The full text is canonical at:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful, be constructive, focus on the work. We expect
contributors to act with good faith and to assume good faith in others.
Unacceptable behavior, harassment, or abuse will be addressed by the
maintainers.

## Reporting

If you experience or witness behavior that violates the Contributor Covenant,
contact the maintainers via a private channel (open a GitHub issue with the
`conduct` label, or email a maintainer directly). Reports will be handled
confidentially. Maintainers commit to acknowledging reports within 7 days.

## Scope

This Code of Conduct applies to all project spaces (code repositories, issue
trackers, discussion forums) and to public spaces where an individual is
representing the project.

## Enforcement

The maintainers will review and investigate reports, then take appropriate
corrective action. Consequences range from a private warning to a temporary
or permanent ban from project spaces, in line with the Contributor Covenant
v2.1 enforcement guidelines.
